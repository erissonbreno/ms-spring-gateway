<h2> Projeto de conclusão de disciplina Arquitetura de Microserviços </h2>
<h3> Micro serviços de aluguel de veículos utilizando API Gateway e Eureka</h3>

Para executar o projeto, executar os serviços: 

**veiculo-service**

**aluguel-service**

**api-gateway**

**eureka-server**

<h4> Documentação </h4>

* Swagger Veiculo: http://localhost:8080/swagger-ui/index.html#/
* Swagger Aluguel: http://localhost:8081/swagger-ui/index.html#/
* Swagger Gateway: http://localhost:9090/swagger-ui/index.html#/
